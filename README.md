# IMDb Movies Data Analysis 

## 📊 Project Overview
This project performs an exploraty data analysis one the IMDb Top 1000 Movies dataset using Python. 

The goal is to explore movie ratings, revenue, genres, and trends over time.

## 🗂 Dataset
Source: Harshit Shankhdhar
Name: IMDB Movies Dataset

## 🔍 Data cleaning and preparation
The dataset was prepared by:
- Removing missing values and duplicates
- Converting columns to correct data types
- Cleaning revenue values and runtime formats
- Stadardizing text columns

## 📈 Exploratory Analysis
The analysis includes:
- Average movie runtime and IMDB rating
- Number of movies released per year
- Top rated movies (Top 10)
- Movies with the most revenue (Top 10)
- Average rating by genre
- Average revenue by genre
- Correlation between ratings and revenue
- Actor performance based on ratings

## 📊 Visualizations
Top project features:
- Top 5 genres by average IMDb rating (bar chart)
- Top 5 genres by average gross revenue (bar chart)
- IMDb rating vs revenue relationship (scatter plot)
- Movie releases over time (line chart)

## 📈 Key Insight
- Action and adventure movies make more money
- Higher rated movies usually earn more revenue
- The number of movies increases over time
- Some actors appear frequently in top rated movies

## 🛠 Technologies Used
- Python
- Pandas
- Matplotlib
